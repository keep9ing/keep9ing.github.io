    📝 부모 컨테이너의 높이 값이 명시적으로 설정되지 않으면 -webkit-fill-available과 같은 값이 올바르게 작동하지 않을 수도

    padding, margin, border 등을 제외한 부모 요소의 사용 가능한 공간 기준으로 자식 요소의 크기를 설정

     WebKit 기반 브라우저(예: Chrome, Safari)에서 지원  표준화된 대체 값으로 stretch 또는 fill을 사용할 수 있다

    .parent {
  width: 500px;
  padding: 20px;
}

.child {
  width: -webkit-fill-available;
  max-width: 300px;
}
