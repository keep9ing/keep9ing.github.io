##### AOP가 뭔데

**AOP(Aspect-Oriented Programming)**는 **관점 지향 프로그래밍**입니다.

핵심 개념: 

비즈니스 로직과 공통 관심사(로깅, 보안, 트랜잭션 등)를 분리하여 코드 중복을 줄이는 프로그래밍 방법입니다.

Spring에서 AOP 사용 예시:

주요 용어:

- @Aspect: AOP 클래스 선언
- @Around: 메서드 실행 전후에 실행
- JoinPoint: AOP가 적용되는 지점
- Pointcut: 어떤 메서드에 적용할지 조건

장점:

- 코드 중복 제거
- 비즈니스 로직과 부가 기능 분리
- 유지보수성 향상
